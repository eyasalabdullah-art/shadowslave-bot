# 🌑 Shadow Slave RPG Bot

بوت تلغرام تفاعلي داخل عالم رواية Shadow Slave

## المتغيرات المطلوبة على Railway:

| المتغير | القيمة |
|---------|--------|
| TELEGRAM_TOKEN | توكن البوت من BotFather |
| GEMINI_API_KEY | مفتاح Gemini API |

## الأوامر:
- `/start` - بدء اللعبة
- `/stats` - إحصائياتك
- `/reset` - إعادة البدء
